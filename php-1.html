PHP

Découverte du language

PHP est un language de script (non compilé) qui peut s'executer en tant que language de script d'un serveur Web (notre cas), comme serveur indépendant (PHP FPM) ou comme executable en ligne de commande (ex : pour faire des scripts).

PHP est un language non typé (pas besoin de déclarer qu'un texte est de type string ou qu'un entier est un int). Sa syntaxe ressemble à celle du C sans la complexité de gestion de mémoire et autre difficultés de ce language.

Un language de script est un language non compilé c'est à dire qu'il est interprêté quand vous lancez son execution.

Dans notre cas nous allons majoritairement faire du PHP utilisé par notre serveur Web (apache).

Dans le serveur que nous avons installé les pages se situent dans /var/www/html/

Créez un fichier test1.php dans lequel nous ferons nos premiers tests.

Apache en découvrant l'extension .php enverra le fichier à son module PHP pour qu'il le parse et renvoi un résultat HTML.

Ce fichier sera très simple et contiendra :

<?php
echo("Je suis un premier script PHP");
echo("<h2>Félicitations développeur !</h2>");
?>

Nous vous recommandons d'utiliser WinSCP pour accéder aux fichiers de votre serveur Linux.
Si vous configurez votre WinSCP pour lancer Sublime Text, vous aurez une chaîne complète de création pour vos développements.
