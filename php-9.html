Fonctions

Jusqu'a présent nous avons créé nos morceaux de programmes directement. Hors dans un environnement "réel" il est impossible de mettre tout le code comme ceci.

Nos programmes vont devoir se découper en fonction, qui nous permettront de rendre notre code plus lisible, modulaire et réutilisable.

Une fonction est définie par

  • Son nom : le nom qui sera utilisée pour l'appeler
  • Ses paramètres : les paramètres qui seront passés à l'appel de la fonction est utilisés dans celle ci
  • Sa valeur de retour : ce que la fonction devra nous retourner
<?php
function direBonjour($compteur){
   for($i=0 ; $i<$compteur ; $i++){
      echo("Bonjour ".$i."<br>");
   }
}

direBonjour(10);
?>

Notre fonction ne retourne rien et affiche le nombre de fois indiquée en paramètre un message.

Essayer maintenant d'appeler la fonction en ommettant le paramètre (direBonjour())

function direBonjour($compteur = 10){
if(gettype($compteur) !== "integer"){
echo("Oula, la valeur du paramètre n'est pas un entier !");
return false;
}
   for($i=0 ; $i<$compteur ; $i++){
      echo("Bonjour ".$i."<br>");
   }
}

direBonjour();
?>

Nous allons maintenant créer une fonction qui additionne deux nombres entier et qui retourne le resultat.

<?php
function additionne($nb1 = false, $nb2 = false){
   if(gettype($nb1) !== "integer" || gettype($nb2) !== "integer"){
      echo("Oula, la valeur des paramètres n'est pas un entier !");
      return false;
   }
   if($nb1 === false || $nb2 === false){
      echo("Oula, il manque des paramètres !");
      return false;
   }
   $result = $nb1 + $nb2;
   return $result;
}

$monResultat = additionne(10, 20);
echo "Resultat de mon addition : ".$monResultat;
?>
