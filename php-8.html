Structures conditionnelles

Boucles

Un programme a généralement besoin d'effectuer des tâches répétitives, comme par exemple changer des valeurs dans un tableau.

for

PHP nous offre plusieurs types de boucles. La première que nous étudierons et la boucle for

<?php
for($i=0 ; $i<100 ; $i++){
   echo("Tour ".$i." dans la boucle<br>");
}
?>

while

Le deuxième type de boucle est le while. On utilisera cette boucle pour effectuer une boucle tant qu'une condition n'est pas respectée.

Attention avec ce type de structure il est facile de faire des boucles infinie qui ne quitte donc jamais et qui peuvent faire "planter" votre serveur.

<?php
$drapeau = false;
$compteur = 0;
while(drapeau === false){
   echo("Tour ".$compteur." dans le while<br>");
$compteur++;
if($compteur == 5)
   $drapeau = true;
}
echo("<h2>On quitte le WHILE</h2>");
?>

foreach

La structure foreach và nous permettre de parcourir par exemple un tableau associatif pour récupérer la clé (le nom du tiroir) et la valeur contenue dans le tiroir.

<?php
$monTableau = Array();
$monTableau['Imane'] = "Je suis Imane !";
$monTableau['Mohammed'] = "Je suis Mohammed !";
$monTableau['Boris'] = "Je suis Boris !";
foreach($monTableau as $key => $value){
   echo("Tiroir ".$key." => ".$value."<br>");
}
?>

switch

Pour éviter l'imbrication de if la structure switch est généralement très utile.

switch permet de prendre une variable est d'effectuer des actions suivant sa valeur.

<?php
$prenom = "Imane";
switch($prenom){
   case "Jeremy":
      echo("Je suis Jeremy");
      break;
   case "Imane":
      echo("Je suis Imane");
   break;
   default:
      echo("Je ne suis pas Imane ni Jeremy");
      break;
}
?>
