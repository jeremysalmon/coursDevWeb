PHP

Manuel PHP

Nous avons déjà utilisé plusieurs fonctions de PHP (print_r, echo, count...). Toutes ses fonctions possèdent des arguments (ce qui se trouvent entre les ()) et parfois une valeur de retour (ex : count nous renvoit un entier qui représente le nombre de lignes dans un tableau).

PHP dispose de plusieurs centaines de fonctions qui vous permettent d'effectuer toutes les manipulations nécessaires à vos programmes.

L'avantage de PHP est que son système d'aide en ligne (en Français) et très complet.

Toutefois il est nécessaire de comprendre sa "philosophie" pour en tirer pleinement profit.

L'idéal si vous connaissez le nom de la commande PHP c'est de taper NOM_COMMANDE php dans Google. Dans la majorité des cas, le premier résultat sera un lien vers la documentation de cette commande.

Sur la page vous aurez la description de la commande

La deuxième partie vous indiquera quels sont les paramètre et la valeur de retour de la fonction

int count ( mixed $array_or_countable [, int $mode = COUNT_NORMAL ] )

Dans la liste des paramètres, tout ce qui est entre [] est optionnel

Vous aurez ensuite une section Liste des Paramètres qui vous détaille l'ensemble des paramètres disponibles (obligatoires et optionnels)

La section suivante vous explique la valeur de retour. Si elle ne retourne rien void est indiqué

La section suivante est un exemple d'utilisaition de la fonction.

Enfin vous avez la partie commentaires qui contient des remarques d'autres développeurs. Cette section peut être utile quand vous rencontrez un problème.
