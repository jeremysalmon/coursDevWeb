Les formulaires

HTML nous permet d'interragir avec les visiteurs à l'aide de formulaires.

Les formulaires sont constitués de balises pouvant représenter des champs textes, des listes déroulantes, des zones de texte long, des cases à cocher ...

Un formulaire se termine généralement par un bouton permettant d'envoyer le formulaire au serveur.

Un formulaire commence par la balise <form> et se termine avec </form>.

La balise ouvrante possède au minimum deux propriétés qui sont :

  • action="xxxxx" : ou xxxx défini l'URL ou le lien vers le fichier qui traitera la demande. Cette URL ou ce lien dirigera généralement vers un script PHP ou autre.
  • method="y" : ou y est soit GET ou POST. y défini le verbe HTML pour envoyer le formulaire.

Le formulaire est ensuite constitué des champs qui constuerons les données à envoyer au serveur.

Pour chaque balise nous mettrons une propriété id et name qui seront utilisées comme référence pour nos données côté serveur.

Les champs à notre disposition :

  • <input type="text" name="xxxx" id="xxxx" /> : champs texte.
  • <input type="checkbox" name="xxxx" id="xxxx" value="zzzz"/> : case à cocher.
  • <input type="password" name="xxxx" id="xxxx"/> : champs texte mais avec les caractères cachés.
  • <input type="hidden" name="xxxx" id="xxxx"/> : champs texte caché permettant de stocker des informations à envoyer au serveur.

La balise <textarea id="xxxx" name="xxxx"></textarea> est une zone de texte long.

La balise <select id="xxxx" name="xxxx"></select> permet de faire des listes déroulantes. Les éléments des listes déroulantes sont des <option value="xxxx">Texte Ici</option>

Enfin la balise <input type="submit" value="Envoyer" /> est le bouton qui enverra les données de votre formulaire vers le script action situé sur le serveur.

Exemple de formulaire

<form action="serveur.php" method="POST">
   <table>
      <tr>
         <td>Champs texte</td>
         <td>
            <input type="text" id="fieldTxt" name="fieldTxt">
         </td>
      </tr>
      <tr>
         <td>Champs password</td>
         <td>
            <input type="password" id="fieldPasswd" name="firldPasswd">
         </td>
      </tr>
      <tr>
         <td>Checkbox</td>
         <td>
            <input type="checkbox" id="fieldCheckbox" name="fieldCheckbox" value="myValue">
         </td>
      </tr>
      <tr>
         <td>Liste déroulante</td>
         <td>
            <select id="fieldSelect" name="fieldSelect">
               <option value="myValue1">ma valeur 1</option>
               <option value="myValue2">ma valeur 2</option>
               <option value="myValue3">ma valeur 2</option>
            </select>
         </td>
      </tr>
      <tr>
         <td>Texte Long</td>
         <td>
            <textarea id="fieldTextarea" name="fieldTextarea"></textarea>
         </td>
      </tr>
      <tr>
         <td colspan="2">
             <input type="submit" value="Envoyer au serveur">
         </td>
      </tr>
   </table>
</form>
