Les variables

Comme nous l'avons indiqué précédemment, PHP est un language non typé. Quand vous créez des variables, vous n'avez donc pas besoin de lui déclarer un type.

Une variable est un "nom" dans laquelle nous voulons mettre une valeur. Cette valeur peut être modifiée durant le cours du programme.

En PHP déclarer une variable se fait :

$maVariable = "texte";
$monChiffre = 12;
$maVirgule = 12.2;

Même s'il n'est pas typé, PHP une fois la valeur attribué à un type.

Pour vérifier le type d'une variable vous pouvez par exemple faire :

echo(gettype($maVariable)."<br>");
echo(gettype($monChiffre)."<br>");
echo(gettype($maVirgule)."<br>");

Si vous imaginez le cas suivant

$prix = "20.2";
$taxe = 5;
echo("Prix : ".gettype($prix)."<br>");
echo("Taxe : ".gettype($taxe)."<br>");
$total = $prix + $taxe;
echo("Total : ".$total. "(".gettype($total).")");

Dans ce cas nous avons fait une addition entre une chaine de caractère ($prix) et un integer ($taxe) et notre résultat est un double (un chiffre a virgule)

PHP nous a "transtypé" nos types pour nous fournir un résultat correct.
