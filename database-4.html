Base de données

MySQL

Durant ce cours, pour nos futurs développements, nous utiliserons le serveur de bases de données MySQL.
MySQL est un serveur de bases de données relationnelles qui peut être utilisé sous Windows, MacOS ou Linux.

Comme tous les serveurs de base de données relationnelles, les interactions avec MySQL se font en SQL. SQL est un language standardisé qui permet de faire des insertions, des modifications, des suppressions dans les tables des bases de données. Le SQL sert également grâce à un syntaxe souple de récupérer les données stockées dans les bases.

Vous pouvez interragir avec MySQL de deux façons :

  • Console MySQL : la console MySQL vous permet de taper toutes les commandes communiquer avec le serveur. La console est intégrée avec toutes les versions de MySQL.
  • Interface Graphique : la plus connue est PHPmyadmin qui permet de faire toutes les interactions avec MySQL de façon graphique.

Pour notre cours nous utiliserons la console MySQL pour mieux comprendre les différentes requêtes que nous utiliserons.
