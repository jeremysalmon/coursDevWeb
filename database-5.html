Installation de votre plateforme

L'installation d'un serveur gérant le serveur Web, PHP et MySQL est très simple sous Debian.

Quand vous êtes en root dans la console tapez :

apt-get install apache2 mysql-server mysql-client php5 libapache2-mod-php5 php5-mysql

Durant l'installation vous aurez à répondre à une question qui vous demandera le mot de passe "root" de la base de données. Pour plus de facilité (mais à ne jamais faire en production) vous pouvez mettre 1234

Votre serveur web est maintenant opérationnel.

Vous pouvez tester le serveur HTTP en lançant un navigateur et en vous connectant à http://VOTRE_IP_LINUX/

Nous vous conseillons fortement de travailler avec Putty, qui est un client SSH et qui vous apporte un maximum de confort pour vos différentes utilisations de la console Linux. Avec Putty vous :

  • Bénéficiez d'"ascenseurs" qui vous permettent de remonter dans l'historique
  • Pouvez copier/coller dans la console
  • Pouvez vous connecter avec plusieurs Putty à la même machine Linux (ex : une fenêtre pour les fichiers de log, une fenêtre pour les commandes, une fenêtre pour éditer des fichiers ...)

Vous pouvez télécharger Putty à l'adresse : http://the.earth.li/~sgtatham/putty/latest/x86/putty.exe
