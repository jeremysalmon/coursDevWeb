Résumons

Bootstrap nous permet donc de n'avoir a écrire aucune ligne de CSS pour avoir des interfaces soignées.

L'aide en ligne très complète nous permet de facilement créer des balises esthétiques.

Modifiez les différents fichiers HTML que vous avez créé précédemment (eleve.html,module.html,examen.html,note.html) en leur appliquant des styles bootstrap.

Créez les pages listEleve.html,listModule.html,listExamen.html,listNote.html qui seront les pages qui listeront les éléments de chaque table.
Mettez pour l'instant des données fictives dans vos tables.
Ajoutez une colonne avec des icones pour supprimer et editer. Pour chaque ligne des tableaux ajoutez ces deux icones.
