Cours Dev Web (Sommaire)

Bienvenue.

L’objectif de ce cours est de vous permettre d'apprendre à développer des applications web.

Pour cela nous :

  • Aborderons les principes de fonctionnement d’une application web
  • Etudierons une méthode classique de gestion d’un projet d’application
  • Utiliserons les différents langages impliqués dans le développement d’applications web : HTML, CSS, Javascript, SQL, PHP …
  • Installerons et configurons nos environnement de développement
  • Développerons une applications web

Dans quelques heures vous serez les heureux créateurs d’une application web
