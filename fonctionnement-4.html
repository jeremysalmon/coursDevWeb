Comment fonctionne une application Web

Base de données relationnelle

  • Le serveur de base de données est le logiciel (service) qui gère l’ensemble des interactions entre les données et les programmes désirant utiliser ces données (ex : MySQL)
  • Une base de données est le « conteneur » des tables. Un serveur de bases de données peut faire fonctionner plusieurs bases de données. Généralement on a une base par application (ex : bddMonApplication, bddBlog, …)
  • Une table représente une entité que l’on veut décrire. Ex : une table utilisateur, une table voiture, une table produit, une table catégorie …
  • Les champs décrivent les caractéristique d’une entité (table). Un champs est constitué d’un nom, d’un type (texte, numérique, date …) et de possibles options. Par exemple pour une table utilisateur, on pourra avoir les champs nom (texte), mail (text), date de naissance (date) …
