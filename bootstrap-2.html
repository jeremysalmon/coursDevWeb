Twitter Bootstrap

Premier Pas

Le principe de Bootstrap est simple. Il suffit d'appliquer des class aux différents éléments de nos pages. Chaque class est un ensemble de code CSS permettant à nos éléments HTML standards d'être "designés" plus esthétiquement.

L'ensemble de la documentation se trouve sur http://getbootstrap.com/

La grille

Le premier principe de Bootstrap est qu'il découpe l'écran en 12 colonnes.

Notre premier travail va consister à modifier la page index.html fournie avec initializr.

Supprimez tout le contenu du body de index.html livré avec initializr.

Nous allons maintenant créer un container et créer nos premières lignes et colonnes.

<div class="container">
   <div class="row">
      <div class="col-md-12" style="background-color:yellow">
         <p>Ceci est une ligne de 12 colonnes</p>
      </div>
   </div>
   <div class="row">
      <div class="col-md-6" style="background-color:red">
         <p>Ceci est une ligne de 12 colonnes</p>
      </div>
   </div>
   <div class="row">
      <div class="col-md-6 col-md-offset-3">
         <p>Marge de 3 colonnes à gaucche</p>
      </div>
   </div>
</div>
