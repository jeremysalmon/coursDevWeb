Base de données

Création d'une base de données

Créer une base de données se fait dans la console de MySQL avec la commande :

CREATE DATABASE maBase

Ou

  • maBase : le nom de la base de données que vous voulez créer

Vous pouvez vérifier que votre base est bien créée en tapant :

show databases;

ATTENTION : dans notre cours nous utiliserons uniquement le compte root pour nous connecter à la base de données. Dans un environnement de production il convient de créer un utilisateur et des droits appropriés sur la base.

Ces permissions se font grâce à la commande GRANT : http://www.journaldunet.com/developpeur/tutoriel/sql/060830-sql-mysql-privileges-droits.shtml
