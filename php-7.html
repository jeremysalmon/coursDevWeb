Structures conditionnelles

If

La première structure conditionnelle que nous rencontrerons dans un programme est le if

If permet d'effectuer une opération SI une condition est validée (ou non)

<?php
$drapeau = true;
if($drapeau === false){
   echo("Le drapeau est FAUX<br>");
   return false;
}
$nom = "SALMON";
if($nom == "SALMON"){
   echo("Bonjour Jeremy<br>");
}
?>

Vous remarquerez qu'a l'intérieur des {} le code n'est pas au bord de la page. C'est l'indendation. Il est très important d'indenter son code correctement pour plus de lisibilité et pour profiter des fonctions de votre éditeur de texte.
Lisibilité : d'un coup d'oeil vous pouvez voir ce qui fait parti de votre if par exemple.
Editeur : Sublime Text par exemple vous permet de "fermer" et d'"ouvrir" les codes indentés.

Il est fortement DECONSEILLE d'imbriquer des if dans votre code. En effet, vous arriverez vite à des codes "spaghetti" difficilement maintenable.
Dans tous les languages il existe des structures conditionnelles qui vous permettent d'avoir à imbriquer des if

Il arrive souvent que vous ayez à tester plusieurs conditions dans un if

Par exemple si l'animal est un CHIEN ET qu'il est de race "CANICHE" alors on fait une action

<?php
$animal = "chien";
$race = "doberman";
if($animal == "chien" && $race == "caniche"){
   echo("Bienvenue caniche");
   return false;
}
echo("Il n'y a que les caniches qui sont autorisés !");
?>

Vous pouvez aussi tester si c'est un CHIEN OU un CHAT

<?php
$animal = "oiseau";
if($animal == "chien" || $animal == "chat"){
   echo("Bienvenue cher animal a quatre pattes");
   return true;
}
echo("Il n'y a que les chiens ou les chats qui sont autorisés !");
?>

Enfin vous pouvez combiner tous ces tests. Attention, ici la précédence (comme en mathématique) s'applique

<?php
$animal = "oiseau";
$race = "moineau";
if(($animal == "chien" || $animal == "chat") || ($animal == "oiseau" && $race == "moineau")){
   echo("Bienvenue cher animal a quatre pattes. (les moineaux aussi !)");
   return true;
}
echo("Il n'y a que les chiens ou les chats qui sont autorisés !");
?>
