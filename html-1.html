HTML

Le HTML est un language de balisage sémantique permettant de décrire la structure de pages Web.

HTML a pour objectif d'être lu et compris par des humains avec un navigateur Web mais également par des programmes comme les crawlers des moteurs de recherche.

Un fichier HTML porte généralement l'extension .html et peut être directement lu par les navigateurs Internet sans passer par un serveur Web.

Comme nous l'avons vu avec PHP, nous pouvons aussi généré du HTML à partir du language de script du serveur.

Le language HTML est constitué de balises qui pour la plupart s'ouvrent et se ferment.

Dans notre cours nous n'étudierons pas l'ensemble des balises existantes mais celles qui sont succeptibles de nous servir dans notre futur projet.

Les balises de formattage

  • <h1>...</h1> : balise de titre. De 1 à 6, 1 étant les titres de plus grande taille.
  • <p>...</p> : Balise de paragraphes.
  • <br /> : Balise permettant d'effectuer un saut de ligne dans une balise <p></p> par exemple.
  • <hr /> : Ligne de séparation verticale.
  • <img src="...."/> : balise permettant d'insérer une image dans la page. La propriété src est obligatoire et pointe vers l'URL complète d'un fichier image (ex : http://www.aptosolutions.co.uk/assets/images/HTML5-and--006.jpg) ou d'un chemin relatif vers une image (ex : img/html-1.jpg).
  • <a href="...."/></a> : balise permettant d'insérer un lien hypertexte. La propriété href est obligatoire et pointe vers l'URL complète d'un fichier (ex : http://www.google.fr/index.php) ou d'un chemin relatif vers un fichier (ex : html-2.html).

Vous pouvez trouver une liste complète des balises HTML sur le site de OpenClassRoom.
