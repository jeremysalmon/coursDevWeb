A vous de jouer !

L'objectif est de remplir votre base de données avec des centaines d'enregistrements.

  • Vous savez créer un tableau associatif. Créez un tableau associatif avec qui contient une "image" des données à insérer.
  • Connectez vous à la base de données en PHP
  • Faites une boucle qui insère tous les enregistrements du tableau dans la base

ASTUCE : vous pouvez faire une boucle pour créer votre tableau associatif et ainsi insérer des centaines d'enregistrements facilement.

Faîtes une deuxieme page qui affiche l'ensemble des lignes de votre table.
