Première connexion à MySQL

Pour vous connecter à MySQL depuis la console, une fois que vous êtes connecté avec Putty à la console de Linux tapez :

mysql -u root -p

Ou

  • u : indique l'utilisateur avec lequel vous voulez vous connecter (root)
  • p : indique que vous devrez saisir un mot de passe

Vous êtes maintenant connecté à la "console" de MySQL. Tapez votre première commande qui servira à lister les bases de données présentes sur le serveur.

show databases;

Pour quitter la console et revenir à Linux tapez :

exit
