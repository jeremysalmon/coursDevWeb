Débugguer son code

Peut être avez vous déjà fait des erreurs dans les exemples précédents. Vous aviez surement une page blanche qui s'affichait. Par défaut votre installation de PHP cache les erreurs aux visiteurs de façon à ne pas afficher de code erreur lors de problèmes sur vos sites.

Pour voir en temps réel vos erreurs, lancez un deuxième Putty sur votre serveur linux.

Lancez la commande tail -f /var/log/apache2/error.log

Vous verrez alors en temps réel le fichier des erreurs de Apache. C'est ici que sont loggués les erreurs de PHP.

Quand vous écrirez du code vous pourrez utiliser la fonction error_log("Ceci est mon message"); pour écrire en temps réel dans le fichier /var/log/apache2/error.log

Ceci vous permettra par exemple d'écrire quand vous passez à un endroit du code, connaitre l'état d'une variable ...
