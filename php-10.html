MySQL

Nous savons maintenant faire des variables, des tableaux, des structures conditionnelles et des fonctions. Nous en avons donc assez pour pouvoir faire des premiers programmes "interressants".

Un programme étant essentiellement destiné à stocker, traiter et restituer des données, nous allons voir comment PHP interragi avec un base de données MySQL.

Connexion à la base de données

<?php
$bdd = mysqli_connect('localhost', 'root', '1234', 'test');
if($bdd === false){
   echo("Impossible de se connecter à la base de données !");
   return false;
}
echo("Nous sommes connectés à la base de données");
?>

A FAIRE : Créez une fonction connectDb(....) qui prend en paramètre le serveur, l'utilisateur, le mot de passe et la base de données. Cette fonction doit retourner true si vous êtes connecté, false sinon.

<?php
function dbConnect($server='localhost',$user='root',$password='1234', $db='test'){
   return mysqli_connect('localhost', 'root', '1234', 'test');
}
?>

Execution de requêtes

Une fois connecté à la base vous allez pouvoir demander à PHP d'executer des requêtes SQL.

L'execution de requête và se passer en 2 étapes :

  • Execution de la requête de PHP vers MySQL
  • Récupération du résultat
<?php
function dbConnect($server='localhost',$user='root',$password='1234', $db='test'){
   return mysqli_connect('localhost', 'root', '1234', 'test');
}

$connexion = $dbConnect();
if($connexion === false){
   echo("Impossible de se connecter à la base !");
   return false;
}

// Execution de la requête
$sql = "SELECT * FROM client";
$resultat = mysqli_query($connexion, '$sql);

// Récupération des résultats
while($donnees = mysqli_fetch_assoc($resultat)){
   echo "Identifiant : ".$donnees['id']."<br>";
}
?>

Fermeture de la connexion

Une fois vos requêtes terminées, le plus "propre" est de libérer la mémoie des résultats. Pour celà il vous suffit d'executer :

mysqli_free_result($resultat);
