Récupérer des données de la table

Notre objectif n'est pas de faire un cours complet de SQL. Nous verrons donc juste de quoi pouvoir récupérer des données suivant différents critères.

La commande SQL pour récupérer des enregistrements est la command SELECT

Par exemple pour récupérer tous les enregistrements de la table firstTable on tapera

SELECT * FROM firstTable;

Pour récupérer tous les enregistrements qui ont un id 1 on tapera

SELECT * FROM firstTable WHERE id = 1;

Pour récupérer tous les enregistrements qui ont une fieldDate entre le 01 janvier 2015 et le 01 février 2015 on tapera

SELECT * FROM firstTable WHERE fieldDate BETWEEN '2015-01-01 00:00:00' AND '2015-02-01 23:59:59';

Pour récupérer tous les enregistrements qui ont un id égal à 1 ou 2 ou 3

SELECT * FROM firstTable WHERE id = 1 OR id = 2 OR id = 3;

Pour récupérer tous les enregistrements qui ont une fieldDate entre le 01 janvier 2015 et le 01 février 2015 et un fieldFloat > 2.5

SELECT * FROM firstTable WHERE fieldDate BETWEEN '2015-01-01 00:00:00' AND '2015-02-01 23:59:59' AND fieldFloat > 2.5
