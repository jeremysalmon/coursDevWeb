Les tableaux

Dans les développements nous aurons souvent besoin de tableaux. Un tableau est un élément qui regroupe un ensemble de valeurs.

On peut comparer un table à une grande armoire. Chaque tiroir contient une valeur. On peut à tout moment récupérer une valeur grâce au numéro de tiroir. On peut grâce à ce numéro modifier la valeur contenu dans un tiroir.

En php un tableau est un Array

Le code ci dessous créé et joue avec un tableau

<?php
$monTableau = Array();
$monTableau[] = "Bonjour";
$monTableau[] = "Bienvenue";
$monTableau[] = "Au revoir";

echo "Tiroir 1 : ".$monTableau[0]."<br>";
echo "Tiroir 3 : ".$monTableau[2]."<br>";
echo "Nombre de tiroirs : ".count($monTableau);
?>

Vous remarquerez que

  • On ajoute à un tableau grâce à []
  • Le premier tiroir porte le numéro 0
  • On peu compter le nombre de "tiroirs" d'un tableau grâce à la fonction count()

Tableaux Multi Dimensionnel

Le tableau précédent avait une seule dimension. Il arrive souvent qu'on ai besoin de plusieurs dimensions comme par exemple faire une evolution de valeurs.

PHP nous permet d'encapsuler plusieurs tableaus.

<?php
$monTableau = Array();
$monTableau[] = Array();
$monTableau[0][0] = "Val 1";
$monTableau[0][1] = "Val 2";
$monTableau[0][2] = "Val 3";
$monTableau[] = Array();
$monTableau[1][0] = "Dimension 2 Val 1";
$monTableau[1][1] = "Dimension 2 Val 2";
$monTableau[1][2] = "Dimension 2 Val 3";
$monTableau[1][4] = "Dimension 2 Val 4";
$monTableau[1][5] = "Dimension 2 Val 5";
echo("<pre>");
print_r($monTableau);
echo("</pre>"); ?>

La fonction print_r vous permet d'afficher le contenu complet d'un tableau
