Twitter Bootstrap

Librairie CSS

Twitter Bootstrap est une librairie Open Source de Twitter qui permet aux développeurs de rapidement donner une touche "sexy" à leur HTML. Sans aucune connaissance en CSS, le développeur và pouvoir créer des interfaces utilisables sur tout support (responsive).

Twitter Bootstrap est directement téléchargeable sur le site http://getbootstrap.com/

Pour bien débuter, nous allons utiliser un site qui va nous générer une page HTML5 valide accompagnées de Bootstrap et des librairies Javascript qui nous permettront de développer notre application.

Connectez vous sur le site http://www.initializr.com/

Cliquez sur Bootstrao.

Laissez les choix par défaut et cliquez sur le bouton Download It!.

Décompressez le fichier zip à l'endroit que vous désirez.

Uploadez ensuite ce répertoire à la racine de votre serveur Web.

Connectez vous à votre serveur web en indiquant le répertoire de vos fichiers Initializr.
